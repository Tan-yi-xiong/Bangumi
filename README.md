# Bangumi
一个追番和看番的Android应用  

# 说明  
这是一个自学Android新手的练习项目; 
打包好的apk已放在release文件夹下, 感兴趣的朋友可以下载体验一下。  

# 功能
- 视频本地缓存
- 追番
- 主页切换
- 滑动调节亮度, 音量, 进度
- 视频调速
